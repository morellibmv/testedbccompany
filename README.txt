A versão enviada já possui o build.

Requisitos:
- Java 8
- Gradle 3.5 (caso deseje fazer um novo build e/ou rodar os testes)

Instruções:
- Para rodar a aplicação, basta executar o jar em build\libs\TesteDBCCompany-main-1.0.jar .
- Esta implementação aceita apenas entrada de dados pelo arquivo input.txt - que deve estar no diretório onde é realizado o comando para executar o jar. (por exemplo: caso execute direto no diretório onde está o JAR, tomei a providência de deixar uma cópia do arquivo input.txt, caso contrário, um erro será gerado de arquivo não encontrado)
